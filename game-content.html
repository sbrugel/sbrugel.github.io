Game Content

Content for Train Simulator 20XX, including scenarios and patches. The scenarios on the page are released as early as 2019, but I have made content for longer than that (older content has since been taken off the web). For TS-related software, including the scenario readme generator and preload shortener, go to the Projects page.

All README files are within each download. Items are listed by release date.
